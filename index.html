디자인 시안 프로토타입 정의서

1. 디자인 컨셉

직관성과 신뢰성을 극대화한 사용자 중심의 전개
  • 데이터 시각화 중심의 레이아웃

    복잡한 수치와 통계 자료를 사용자가 한눈에 파악할 수 있도록 그리드 시스템 기반의 깔끔한 카드 레이아웃을 적용했습니다.

  • 높은 가독성과 스캐너빌리티(Scannability)

    넓은 여백(White Space)을 적극적으로 활용하여 시각적 피로도를 낮추고, 텍스트 정보와 차트(파이 차트, 바 차트) 간의 균형을 맞춰 정보의 인지 속도를 높였습니다.

  • 현대적이고 전문적인 무드

    불필요한 장식적 요소를 배제하고 플랫하고 모던한 UI 스타일을 채택하여, 관리 시스템에 걸맞은 전문성과 신뢰감을 전달하고자 했습니다.

2. 컬러 시스템

신뢰감을 주는 블루와 깨끗한 화이트의 조화, 직관적인 포인트 컬러 활용
  • 베이스 컬러 (화이트 배경)

    전체 배경을 깨끗한 화이트 톤으로 설정하여 밝고 쾌적한 화면을 연출하고, 콘텐츠 자체에 시선이 가장 먼저 머무를 수 있도록 베이스 역할을 부여했습니다.

  • 포인트 컬러 (블루 계열)

    포인트 컬러는 기업의 상징성과 전문성, 스마트한 이미지를 대변하는 블루 계열을 사용하여 시스템의 일관된 톤을 유지합니다.

  • 서브 컬러 (오렌지, 그린, 옐로우)

    오렌지 계열의 서브 컬러를 중심으로 그린과 옐로우 컬러를 전략적으로 배치하여, 전체적인 블루 톤 데이터의 다양성을 명확하게 구분할 수 있도록 설계했습니다. 계약 유형 비중이나 진행 상태 등 서로 다른 속성의 데이터를 시각적으로 분리하여 인지 오류를 줄이는 역할을 합니다.

  • 보더 및 가독성 대비 (그레이 & 차콜)

    전체 영역 및 카드 영역의 경계선이나 배경에는 부드러운 그레이 톤을 적용하고, 텍스트에는 명도 대비가 명확한 차콜(어두운 그레이) 컬러를 사용하여 눈의 부담을 최소화하면서 가독성을 최대로 높였습니다.

3. 레이아웃 구성

사용성 및 확장성을 고려한 UI/UX 설계
  • 상단 (Header)

    시스템명 및 현재 로그인한 사용자 정보를 제공하며, 권한그룹 변경 제어 장치를 배치하여 사용자가 임의로 권한 역할군을 오가며 효율적으로 기능을 관리할 수 있도록 설계했습니다.

  • 왼쪽 메뉴 (LNB)

    좌측 사이드바 구조로 1Depth와 2Depth 아코디언 메뉴를 설계하였으며, 화면을 더 넓게 쓸 수 있도록 숨김 가능한 접기/펴기 기능 및 하단부 유관 시스템 바로가기 기능을 제공합니다.

  • 브레드크럼 (Breadcrumb)

    현재 화면 위치(예: 공통관리(설정) > 관리자 관리)를 쉽게 인지할 수 있는 경로 표시 영역을 체계적으로 표기했습니다.

  • 콘텐츠 영역 (Contents)

    일관된 여백(Padding) 정책을 통해 그리드, 폼, 카드가 촘촘하면서도 짜임새 있게 묶이도록 배치했으며, 논리적 구조에 따른 오브젝트 레이아웃과 실제 업무 시나리오 기반 설계를 충실히 반영했습니다.

  • 인터랙션 적용 계획

    컴포넌트 단위의 디테일과 트랜지션을 고려하여, 모든 버튼과 인터랙티브 요소에 부드러운 호버 상태 변화 효과를 적용할 계획입니다.
